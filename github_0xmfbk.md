## 🧠 My approach to web app pentesting

| Layer | What I actually do |
|-------|--------------------|
| **Business Logic** | Parameter tampering beyond classic flaws — quota, workflow, race conditions |
| **Authentication** | Session prediction, JWT misuse, OAuth logic bypass |
| **API Security** | Mass assignment, broken object-level auth, improper state handling |
| **Hidden surfaces** | Unintended entry points in GraphQL, WebSockets, SSR |

> 🔍 No scanner finds these. Only a researcher who reads between the lines.
